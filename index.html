Background

SIVA is a Discord bot. That much should be obvious. The name "SIVA" came from the game Destiny, in the Rise of Iron DLC addon. I do not own the logo, it was created by Bungie. The main goal of SIVA is to be the best bot it can be, with my limited knowledge of C# and Discord.NET. I started SIVA rewrite (from Python to C#) early February 2018, and have been going ever since. The source code is not available and it may never be. The most I'll ever release is the Dependencies and the actual bot executable so you can run it for yourself although that may never happen.
